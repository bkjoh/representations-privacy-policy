# Representations for Windows Privacy Policy

Representations for Windows is an offline reference app for number bases, Unicode scalar values, and UTF-8 bytes.

The Windows app does not collect, store, transmit, sell, or share personal information. It does not require an account, does not include advertising, and does not require network access for app features.

Search queries and copied values remain on the local device. The app reads its bundled SQLite reference database locally.

For questions, use the support contact listed with the app store listing.
